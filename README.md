# Ravindra-IAS-couching
Ravindra IAS couching is a project made for frontend battle 2026 by IEEE CS, it is purely frontend with no backend or support.
🚀 Ravindra IAS Landing Page

A modern, responsive, and interactive landing page for a UPSC coaching institute.
Built with a focus on clean UI/UX, performance, accessibility, and modular code structure.

📌 Features
⚡ Fully responsive design (mobile + desktop)
🎯 Clean and modern UI with structured layout
📊 Animated counters and scroll-based reveal effects
📱 Mobile-friendly navigation with hamburger menu
🎨 Custom cursor and interactive canvas background
📋 Form validation with user feedback
❓ FAQ toggle system
🔍 SEO optimized meta tags
♿ Improved accessibility using ARIA attributes
🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (Vanilla)
Scripting: Python (for automation & code structuring)
📂 Project Structure
├── ravindra-ias.html   # Main HTML file (structure + content)
├── styles.css          # Styling and UI design system
├── main.js             # Interactivity and animations
├── extract.py          # Extracts inline CSS/JS into separate files
├── semantic_update.py  # Improves semantic HTML and accessibility
⚙️ How It Works
1. Base Website

The project starts as a single HTML file with inline CSS and JavaScript.

2. Code Modularization

Run:

python extract.py
Extracts <style> → styles.css
Extracts <script> → main.js
3. Semantic Improvements
python semantic_update.py
Adds semantic structure (<header>, <main>)
Improves accessibility (ARIA attributes)
Enhances JS behavior for better UX
